# 更新日志

## [v1.0.1] - 2026-03-14

### 新增功能
- **组件拆分与重组**：将 ControlPanel 拆分为 SizeControlPanel（尺寸设置）和 AdvancedControlPanel（高级选项），优化布局结构
- **图片编辑实时预览**：添加防抖处理，亮度、对比度等参数调整后可实时预览，滑动更流畅
- **高级选项预览按钮**：在高级设置面板添加"更新预览"按钮，不占用生成图纸次数
- **色板颜色预览**：在颜色选择下拉框中添加颜色块预览，直观显示每个颜色效果
- **全新导出图片格式**：创建 patternExport.ts，实现专业拼豆图纸排版效果

### 优化改进
- **简化导出选项**：从多个选项简化为两个：带编号的图片 / 不带编号的图片
- **优化图纸排版**：
  - 顶部和底部显示列编号，左右两侧显示行编号
  - 图案区域带网格线，清晰显示拼豆图案
  - 底部显示完整颜色列表（每行4个，格式：颜色代码 + 数量）
  - 移除标题，界面更简洁
- **调整按钮位置**：将"生成图纸"按钮从顶部移到尺寸设置面板底部

### 问题修复
- **修复图纸缩放功能**：移除缩放限制，现在可以正常放大预览图
- **修复图片编辑参数无反应**：添加 useEffect 监听器，参数变化自动应用到图片
- **修复用户登录问题**：
  - 添加缺失的 device_type 字段到 UserLogin schema
  - 更换密码哈希算法为 pbkdf2_sha256，解决环境兼容性问题
  - 配置前端 API 地址（VITE_API_URL）
- **修复导入路径错误**：修正 AdvancedControlPanel 中的 patternGenerator 导入路径

## [v1.0.0] - 初始版本

### 核心功能
- 图片上传与编辑（亮度、对比度、饱和度调整）
- 拼豆参数设置（颗粒尺寸、目标尺寸、边缘平滑度、颜色容差等）
- 实时预览画布，支持缩放和网格显示
- 色板选择与颜色替换
- 图纸生成与导出
- 用户注册/登录系统
- 试用次数管理
