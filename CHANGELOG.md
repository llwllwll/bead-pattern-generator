# 更新日志

## [v1.0.5] - 2026-03-19

### 新增功能
- **管理员管理后台**：
  - 添加管理员列表展示，支持查看所有管理员信息
  - 添加管理员创建功能（超级管理员可创建超级管理员和普通管理员，普通管理员只能创建普通管理员）
  - 添加管理员密码修改功能（无需输入旧密码，直接设置新密码）
  - 添加管理员删除功能（仅超级管理员可删除普通管理员）
  - 在管理后台顶部显示当前登录管理员的用户名和角色标签
- **用户管理增强**：
  - 添加用户删除功能
  - 添加用户搜索功能，支持按用户名和邮箱搜索
- **激活码激活功能**：用户登录后可在用户信息区域输入激活码进行激活
- **管理员信息获取**：登录后自动获取并显示管理员详细信息

### 优化改进
- **新用户默认额度**：从 10 调整为 5
- **邮箱处理**：用户注册时，如未提供邮箱则保持为空，不再填充占位数据
- **路由配置优化**：统一 API 路由前缀，移除重复前缀
- **401 错误处理优化**：在数据加载函数中遇到 401 未授权错误时，正确清除 token 并更新状态，自动跳转到登录界面

### 问题修复
- **修复管理员身份显示问题**：修复超级管理员登录后显示"管理员"而非实际用户名的问题
- **修复管理员权限控制**：
  - 普通管理员只能修改自己的密码
  - 普通管理员无法删除其他管理员
  - 超级管理员可以修改所有管理员密码和删除普通管理员

## [v1.0.4] - 2026-03-18

### 新增功能
- **色彩库两级菜单结构**：品牌 → 系列 → 颜色的三级结构，替代原来的扁平化色库
- **品牌管理后台**：支持添加、编辑、删除品牌，批量导入品牌数据
- **系列管理后台**：支持添加、编辑、删除系列，按品牌筛选
- **颜色管理后台**：支持添加、编辑、删除颜色，查看颜色属性
- **批量导入功能**：支持 CSV 文件上传解析，预览后确认导入
- **拖动排序功能**：品牌和系列列表支持拖动调整显示顺序
- **批量操作功能**：支持多选删除和批量导出数据

### 优化改进
- **移除默认数据缓存**：页面加载时自动从后端获取最新数据，无需手动刷新
- **数据库结构优化**：删除旧版 color_palettes 和 palette_colors 表，使用新的 brands/series/colors 结构
- **后端 API 优化**：添加批量排序、批量删除、数据导出等接口
- **前端状态管理**：更新 Zustand store 适配新的数据结构

### 问题修复
- **修复 500 错误**：解决 SQLAlchemy 异步懒加载问题，预加载品牌-系列关联关系
- **修复 Select 组件警告**：确保选项 label 和 value 一致性

## [v1.0.1] - 2026-03-14

### 新增功能
- **组件拆分与重组**：将 ControlPanel 拆分为 SizeControlPanel（尺寸设置）和 AdvancedControlPanel（高级选项），优化布局结构
- **图片编辑实时预览**：添加防抖处理，亮度、对比度等参数调整后可实时预览，滑动更流畅
- **高级选项预览按钮**：在高级设置面板添加"更新预览"按钮，不占用生成图纸次数
- **色板颜色预览**：在颜色选择下拉框中添加颜色块预览，直观显示每个颜色效果
- **全新导出图片格式**：创建 patternExport.ts，实现专业拼豆图纸排版效果

### 优化改进
- **简化导出选项**：从多个选项简化为两个：带编号的图片 / 不带编号的图片
- **优化图纸排版**：
  - 顶部和底部显示列编号，左右两侧显示行编号
  - 图案区域带网格线，清晰显示拼豆图案
  - 底部显示完整颜色列表（每行4个，格式：颜色代码 + 数量）
  - 移除标题，界面更简洁
- **调整按钮位置**：将"生成图纸"按钮从顶部移到尺寸设置面板底部

### 问题修复
- **修复图纸缩放功能**：移除缩放限制，现在可以正常放大预览图
- **修复图片编辑参数无反应**：添加 useEffect 监听器，参数变化自动应用到图片
- **修复用户登录问题**：
  - 添加缺失的 device_type 字段到 UserLogin schema
  - 更换密码哈希算法为 pbkdf2_sha256，解决环境兼容性问题
  - 配置前端 API 地址（VITE_API_URL）
- **修复导入路径错误**：修正 AdvancedControlPanel 中的 patternGenerator 导入路径

- 色板选择与颜色替换
- 图纸生成与导出
- 用户注册/登录系统
- 试用次数管理
